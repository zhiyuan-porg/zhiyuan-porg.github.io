# Listener、Ajax、JSON

# 1. Listener 监听器

​ 在 JavaWeb 中的监听器分为三大类（监听的对象）、八大种（监听的属性）。

三大类： 监听域对象（域对象的生命周期）、监听属性（值的变化）、 监听 Session 的状态

（1）监听域对象

对象类型 监听器
ServletContext ServletContextListener
HttpSession HttpSessionListener
HttpServletRequest ServletRequestListener

Warning：

​ 在使用 HttpSessionListener 监听器或 ServletRequestListener 监听器时，如果是直接关闭服务器来销毁这二者时，并不会触发相应的 Destroyed 方法。（这里有一个很俗的例子，将监听器比作一位工作者，服务器关闭时可以看成是下班，那都下班了还管 der）

（2）监听属性变化

对象类型 监听器
ServletContext ServletContextAttributeListener
HttpSession HttpSessionAttributeListener
HttpServletRequest ServletRequestAttributeListener (监听移除)

Warning: 在监听属性的变化时不仅仅是监听值的替换操作，还有添加和移除操作。

（3）监听 Session 对象的状态

监听器 功能
HttpSessionBindingListener 监听 HttpSession 的绑定与解绑
HttpSessionActivationListener 监听 HttpSession 的钝化与活化

# 2.AJAX

​ AJAX（Asynchronous Javascript And XML， 异步 JavaScript 和 XML），一种创建交互式网页应用的网页开发技术。实现了异步请求的功能，即可以在不刷新网页的情况下重新渲染页面的内容。在经过长久的发展之后，AJAX 也有比较完善的封装，下文就以 JQuery 框架下的 AJAX 为进行说明。

​ 在 JQuery 框架下的 AJAX 中主要有 GET、POST、AJAX 三种请求方式。如下表：

请求方式 语法 说明
GET 请求 $.get(url, [data], [callback], [type]) 简单的 get 请求
POST 请求 $.post(url, [data], [callback], [type]) 简单的 post 请求
AJAX 请求 $.ajax([settings]) 复杂的 ajax 请求（可以设更多的参数）
GET 请求 $.get([settings]) 复杂的 get 请求（可以设更多的参数）
POST 请求 $.post([settings]) 复杂的 post 请求（可以设更多的参数）
