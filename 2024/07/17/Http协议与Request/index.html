# Http 协议与 Request

# Http 概述

# 概念

HTTP（Hyper　Text　Transｆｅｒ　Ｐｒｏｔｏｃｏｌ超文本传输协议） 应用层协议，运行在ＴＣＰ传输层协议之上，作用是指定传输格式，且通信双方都要遵守。

Tip：HTTP 协议的默认端口是８０哦

# 特点

由请求报文和响应报文组成；无状态协议，不记录用户访问状态。

# 组成

请求行、请求头、请求体

请求行：      包括请求的方式（GET、POST、PUT、DELETE 等）、地址（URL）以及协议的版本

POST 方式 GET 方式
地址栏 参数不会在地址栏显示出来 参数会拼接在请求的 URL 后面，以查询字符方式
大小 理论上无大小限制，使用流的方式发送数据 受浏览器限制，通常是１K
安全性 在请求体中发送，参数不外显，安全性相对来说较高 在请求行中发送，参数外显，安全性较低
缓存 不使用缓存 可能会使用缓存访问静态资源，使用时状态码是３０４
