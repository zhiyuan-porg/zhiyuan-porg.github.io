# Response And ServletContext

# 1.Http 响应概述

# 1.1HTTP 响应信息的组成

响应行、响应头、响应体

1.2.1 响应行

由协议及版本、响应状态码、状态信息

e.g. HTTP/1.1 200 ok

其中，响应状态码又分为五类：

状态码 含义
1xx 指示信息；表示请求已接收，继续处理
2xx 成功；
3xx 重定向
4xx 客户端错误；语法错误 or 无法实现
5xx 服务端错误

tips: 常见的状态码

200：正确得到响应数据

302：页面重定向

304：使用了本地缓存

400：请求参数错误

404：资源不存在

405：请求的方法错误

415： 无法处理的媒体格式

500： 服务器内部错误

1.2.2 响应头

由各种键值组成，键固定值可变。常见的响应头如下：

响应头信息 说明
Location 重定向要跳转的地址
Content-Type 设置数据类型及字符集
Refresh 设置稍后跳转的延迟时间及页面路径
Content-Disposition 设置内容的处理方式

1.2.3 响应体

封装服务器要返回的数据，而且可以是任意数据

tips: Get 请求虽然没有请求体，但它有响应体嘞
